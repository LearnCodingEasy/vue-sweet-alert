# vue-sweet-alert

# Install
```
npm i vue-sweetalert2
```
# IMPORT
```
import VueSweetalert2 from "vue-sweetalert2";
```
```
import "sweetalert2/dist/sweetalert2.min.css";
```
# Use
#### Html
```
<button @click="showAlert">Hello world</button>
```
#### Javascript Function 
```
showAlert() {
    // Use sweetalert2
    this.$swal("Hello Vue world!!!");
},
```
